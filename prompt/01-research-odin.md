# Tahap 1 — Riset Odin Card Game

## Prompt asli
> saya mau kamu untuk buat website game, nantinya akan ada banyak game dan
> bisa diberi opsi seperti ada username sebelum main, score, mode
> single/bot/multiplayer, dll. petama saya mau kamu mengumpulkan informasi
> dulu tentang game Odin Card Game. Coba kumpulkan informasi tentang
> gamenya dulu

## Konteks
Sebelum membangun apa pun, kumpulkan dulu aturan lengkap game kartu fisik
"Odin" (Helvetiq) lewat pencarian web — supaya implementasi digitalnya
akurat, bukan tebakan.

## Yang dilakukan
- `WebSearch` + `WebFetch` ke beberapa sumber (BoardGameGeek, Games Night
  Guru, What Board Game, Helvetiq resmi, Opinionated Gamers) untuk
  memastikan detail mekanik: 54 kartu 6 warna angka 1–9, kombinasi
  digit-tersusun-turun untuk membentuk nilai, mekanik "ambil kartu yang
  dikalahkan", kondisi menang/skor.
- Tidak ada kode yang ditulis di tahap ini — murni riset & ringkasan.

## Hasil
Ringkasan aturan lengkap disampaikan ke user sebelum lanjut ke tahap
implementasi (Tahap 2).
